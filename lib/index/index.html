Графіки

Граф показує можливість використання E-Declaration system як аналітичної системи.
Графіки показують реальній розподіл доходу згідно поданих декларацій з карєрним ростом.

- Прокурор -> PRSC
- Перший замісник прокурора -> FDP
- Другий замісник прокурора -> SDP
- Начальник місцевого відділу прокуратури -> HLPO
